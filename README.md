# WT_5w0
Semantic Elements, CSS3 &amp; Web Storage API Demo  This project demonstrates the use of HTML5 semantic elements, CSS3 styling, Tailwind CSS utilities, media-responsive design, and the Web Storage API to build a simple, structured, and interactive webpage.
Features 
Uses semantic HTML elements such as header, main, section, article, aside, and footer
Styled with Tailwind CSS and custom CSS
Responsive layout using modern CSS techniques
Numeric list styling using Tailwind utility classes
Theme switcher implemented using Web Storage API (localStorage)
Clean and readable UI with accessibility-friendly structure
Technologies Used
HTML5
CSS3
Tailwind CSS
JavaScript
Web Storage API 
Purpose 
This project is created as part of a Web Technologies Lab to understand semantic markup, modern CSS frameworks, and client-side storage concepts.
